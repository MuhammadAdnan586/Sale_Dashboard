# Sales Analytics Dashboard - Power BI

Interactive sales dashboard for FY2024 built in Power BI.

## Key Metrics
- Total Revenue: 11M PKR
- Total Profit: 3M PKR  
- Profit Margin: 29.59%
- Total Orders: 87

## Features
- Monthly revenue trend
- Regional sales comparison
- Category breakdown
- DAX measures for KPIs
- Interactive slicers

## Tools Used
Power BI Desktop, DAX, CSV Data Modeling
