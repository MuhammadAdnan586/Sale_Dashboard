### 📌 About the Project

Interactive **sales dashboard for FY2024** built in Power BI — gives a quick, visual breakdown of revenue, profit, and order performance across regions and categories, so business decisions don't need to wait on a spreadsheet.

---

### 📊 Key Metrics

| Metric | Value |
|---|---|
| 💰 Total Revenue | **11M PKR** |
| 📈 Total Profit | **3M PKR** |
| 📐 Profit Margin | **29.59%** |
| 🧾 Total Orders | **87** |

---

### ✨ Features

- 📅 Monthly revenue trend
- 🌍 Regional sales comparison
- 🗂️ Category breakdown
- 🧮 DAX measures for KPIs
- 🎛️ Interactive slicers

### ⚙️ Installation

```bash
# 1. Clone the repo
git clone https://github.com/MuhammadAdnan586/Sale_Dashboard.git
cd Sale_Dashboard

# 2. Open the .pbix file in Power BI Desktop
"sale dashboard.pbix"
```

> Requires [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free) to open and explore the report.
